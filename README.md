# Snake Game with AI

Welcome to the Snake Game project, a classic arcade game reimagined with AI capabilities for automated gameplay. This project focuses on creating a simple yet engaging experience that can be played both manually and autonomously by an AI.

## Project Summary

The goal of this project is to develop a classic Snake game that can be played manually or automatically using AI rules. The game is designed to be lightweight and run smoothly in a web browser without the need for a database or complex graphics.

## Goals
- **Functional Snake Game**: Create a playable version of the Snake game.
- **AI Automation**: Implement AI rules to enable autonomous gameplay.
- **Lightweight and Efficient**: Ensure the game is lightweight and performs well in a web browser.

## Non-Goals
- Multiplayer functionality.
- Complex graphics or animations.
- Database integration for scores or game states.

## Tech Stack
- **Frontend**: HTML5 Canvas and JavaScript
- **Game Engine**: None
- **AI Logic**: JavaScript
