# Feature List

This document provides a detailed list of features included in the Snake Game with AI project.

## MVP Features

1. **Basic Snake Gameplay**
   - Core mechanics where the snake grows longer as it eats food.
   - Game ends if the snake hits the wall or itself.
   - Priority: Must

2. **AI Player**
   - AI rules that allow the snake to play autonomously.
   - Decision-making to avoid obstacles and eat food.
   - Priority: Must

3. **Simple UI**
   - Basic user interface with start, pause, and reset game options.
   - Priority: Should

4. **Score Display**
   - Displays the current score based on the number of food items eaten.
   - Priority: Should

5. **Responsive Design**
   - Ensures the game is playable on both desktop and mobile browsers.
   - Priority: Could

## Future Features

- Explore additional AI complexity.
- Consider visual enhancements or additional game modes.

This feature list is intended to guide development and may evolve based on feedback and project direction.
